# v0.1.0 Release Notes

## 概览
v0.1.0 是“最小可跑通”的 Java Agent 脚手架版本：先把端到端链路跑通，再在后续版本逐步增强能力。

## 这次增加了什么（What）
- Maven 打包为可运行 jar（CLI 入口）。
- OpenAI 兼容 `/v1/chat/completions` 客户端（`OpenAiCompatClient`）。
- JSON Memory：对话历史落盘到 `memory.json`（`JsonMemoryStore`）。
- 工具调用（prototype）：模型输出 JSON 触发工具执行（`time.now` / `http.fetch`）。

## 为什么要增加（Why）
- 先验证“能跑、能连、能对话、能调用工具、能记忆”的基本闭环，避免一开始就引入复杂框架导致调试成本高。

## 解决了什么问题（Problem Solved）
- 有了一个可以立即运行、可调试、可扩展的最小 Agent 基座，后续新增能力（更多工具、规划器、路由、多模型等）都有落点。

## 设计与实现要点（How）
- 以 `Main` 的 while 循环作为交互入口，核心逻辑集中在 `Agent.respond()`。
- 工具调用通过 `ToolCall.tryParse()` 解析“纯 JSON 输出”触发，尽量保持协议简单。

## 使用方式（Usage）
```bash
mvn -q -DskipTests package
java -jar target/java-agent-0.1.0.jar
```
